Detailed Grid System Comparison

Bootstrap 3 Grid System

Bootstrap 3 uses a 12-column fluid grid with classes such as .col-md-*, .col-sm-*, and .col-xs-*. It is not based on Flexbox. Offsets are applied using classes like .col-md-offset-*.

.col-md-4
.col-md-4 .col-md-offset-4

.col-md-6
.col-md-6

.col-md-4

In the above example, note how the offset (.col-md-offset-4) pushes the column to the right.

Bootstrap 3 Code Example

    <div class="row">
    <div class="col-md-4">...</div>
    <div class="col-md-4 col-md-offset-4">...</div>
    </div>

    <div class="row">
    <div class="col-md-6">...</div>
    <div class="col-md-6">...</div>
    </div>

Bootstrap 4 & 5 Grid System

Bootstrap 4 and 5 use a Flexbox-based grid system with similar classes (.col-*, .col-md-*, etc.) and improved responsiveness. Offsets now use the .offset-md-* classes.

.col-md-4
.col-md-4 .offset-md-4

.col-md-6
.col-md-6

.col-md-8
Nested .col-md-6
Nested .col-md-6
.col-md-4

In Bootstrap 4/5, the offset is defined as .offset-md-4 instead of .col-md-offset-4. The Flexbox-based grid allows for more flexible alignment and ordering of columns.

Bootstrap 4/5 Code Example

    <div class="row">
    <div class="col-md-4">...</div>
    <div class="col-md-4 offset-md-4">...</div>
    </div>

    <div class="row">
    <div class="col-md-6">...</div>
    <div class="col-md-6">...</div>
    </div>
